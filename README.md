# 実用 Go言語

- 使用する技術書は「実用 Go言語」(オライリー・ジャパン刊行)
  - https://www.oreilly.co.jp/books/9784873119694/
  - ![画像](https://www.oreilly.co.jp/books/images/picture_large978-4-87311-969-4.jpeg)
